# bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!

BookBot is a Python command-line application developed as my first project while completing the Boot.dev Backend Development curriculum.

## Features

- Reads text files from the command line
- Counts total words
- Counts the frequency of each character
- Generates a formatted analysis report
- Supports analyzing different books through command-line arguments

## Technologies Used

- Python
- Command Line Interface (CLI)
- Visual Studio Code
- Git
- GitHub
